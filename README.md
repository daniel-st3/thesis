# Thesis Project: Political Bias & Retail Sentiment in Short-Term Stock Forecasts

## Setup
1. `conda env create -f environment.yml`
2. `conda activate thesis_env`

## Folder Structure
- `data/` – raw & processed datasets  
- `notebooks/` – EDA notebooks  
- `src/` – Python scripts for ingestion, feature engineering, modeling  
- `reports/` – figures and exported results
